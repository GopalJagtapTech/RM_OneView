# RM OneView Automation Framework

Automation framework created for RM OneView application used in NBFC domain.

## Objective

To automate relationship manager workflows such as:

* Customer search
* Deposit details validation
* RM mapping verification

## Technologies Used

* Java
* Selenium WebDriver
* TestNG
* Maven

## Framework Features

✔ Page Object Model
✔ Reusable utilities
✔ Automated validation checks
✔ Structured test execution

## Test Modules

* Customer Search
* Deposit Details
* RM Mapping

## Execution

Run the TestNG runner class to execute automation tests.
